# 记录详情图表说明

## 详情页里有什么

记录详情页会展示：

- 时间范围与时长
- 平均功率
- 电量变化
- 亮屏 / 息屏时长
- 功耗、电量、温度相关图表

## 底部图例可以点击

图表底部有多个图例，其中前三个可以直接点击：

- 功率
- 电量
- 温度

后面的“亮屏”和“息屏”图例只是说明颜色，不负责切换显示。

## 功率图例的行为

点击功率图例时，会按下面的顺序循环切换：

`原始曲线 -> 趋势曲线 -> 隐藏`

## 电量和温度图例的行为

- 点击“电量”图例，可以隐藏或恢复电量曲线
- 点击“温度”图例，可以隐藏或恢复温度曲线

如果你只想看某一条线，先把其他图例点掉就行。

## 什么时候适合切换图例

- 想观察瞬时波动：优先看 `Raw`
- 想看整体走势：切到 `Fitted`
- 想减少干扰：隐藏暂时不关心的曲线
